### Reporting & Insights

By the end of Week 1 you should be able to pull a basic performance read on **{{account}}**
and say something useful about it — not just restate the numbers.

- Start from the shared {{account}} dashboard; don't build one-off spreadsheets.
- Every insight should answer "so what?" — a number without a recommendation isn't an insight.
- Flag anything that looks off to **{{manager}}** early; surfacing a problem is never a bad look.

To go deeper on the metrics and the data model behind them, you'll work through the
analytics deep-dive — see that module for the full picture.
