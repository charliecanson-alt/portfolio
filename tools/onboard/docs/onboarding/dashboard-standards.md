### Dashboard Standards

A dashboard nobody trusts is worse than no dashboard. These are the standards for anything you publish on **{{account}}**.

- One dashboard, one question — don't cram five stories onto one screen.
- Every metric needs a clear definition and a date range; ambiguity erodes trust fast.
- Label units, annotate anomalies, and link back to the source so others can verify.

To build these well you'll lean on the analytics deep-dive — see that module for the metric definitions and data model.
