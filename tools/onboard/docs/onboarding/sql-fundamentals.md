### SQL Fundamentals

As an **{{role}}** you'll live in the data, so get comfortable querying it directly
rather than waiting on someone else to pull numbers.

- Learn the core {{account}} tables and how they join — start from the documented views.
- Write `SELECT` queries before you write `JOIN`s; get one table right first.
- Save useful queries somewhere shared so the next analyst on **{{account}}** isn't starting cold.

If a query looks expensive or you're unsure of a table, ask **{{manager}}** before running it.
